This repository contains a starter project for the AIMMS-MOPTA Optimization Modeling Competition 2026.

AIMMS_MOPTA_2026 is the project folder, containing a .aimms file, data contained in an excel spreadsheet, along with all other requirements to open the AIMMS project.

To get started, academic users can access a free AIMMS license here: https://licensing.cloud.aimms.com/license/academic.htm

Visit the AIMMS Download center to install AIMMS: https://www.aimms.com/support/downloads/

Read the Readme.docx for instructions on how to use the AIMMS project.

For more information about getting started with AIMMS, visit: https://how-to.aimms.com/Articles/363/363-getting-started-aimms.html.
